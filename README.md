# 1. PowerShell Code

1-1. Variable: $
Syntax: $name

1-2. Output: Write-Host
Comparison:

Python: print(f"Hello")
JavaScript: console.log("Hello")
PowerShell: Write-Host "Hello"

1-3. User Input: Read-Host
Syntax: Read-Host "Enter"

1-4. Semicolon: ;

1-5. Example:

$name = Read-Host "Enter your name"
Write-Host "Welcome, $name."

# 2. Basic Commands

2-1. Check file list: ls

2-2. Check current path: pwd

2-3. Create a new folder: mkdir

2-4. Help: help

2-5. Delete folders and files: rm

2-6. Create a text file: echo "Hello" > test.txt

2-7. Open a folder: cd

2-8. Open a file: cat

2-9. Open an exe: ./test.exe

2-10. Open a text file encoded in UTF8: cat "test.txt" -Encoding utf8
